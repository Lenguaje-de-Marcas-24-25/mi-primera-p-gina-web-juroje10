Términos y Descripciones

  1. W3C
    Consorcio World Wide Web, organización que desarrolla estándares para la Web.
  2. WHATWG
    Web Hypertext Application Technology Working Group, grupo enfocado en la evolución de HTML y tecnologías relacionadas.
  3. HTML
    Lenguaje de marcado utilizado para crear y estructurar el contenido de páginas web.
  4. XHTML
    Versión más estricta de HTML que sigue reglas de XML.
  5. DTD
    Document Type Definition, define la estructura y los elementos permitidos en un documento HTML o XML.
  6. APIs
    1. Drag and Drop
      Permite a los usuarios arrastrar y soltar elementos en una página web.
    2. Geolocalización
      API que proporciona la ubicación geográfica del usuario.
    3. Almacenamiento local
      API que permite almacenar datos de manera persistente en el navegador.
    4. App Cache
      Permite a las aplicaciones web almacenar recursos para funcionar sin conexión.
    5. WebWorkers
      Permite ejecutar scripts en segundo plano para mejorar el rendimiento.
  7. Editor WYSIWYG
    "What You See Is What You Get", un editor visual que permite ver el contenido como se verá al publicarse.
  8. Navegadores
    1. Mozilla Firefox
      Navegador de código abierto desarrollado por Mozilla.
    2. Google Chrome
      Navegador rápido y popular desarrollado por Google.
    3. Safari
      Navegador desarrollado por Apple, optimizado para dispositivos Apple.
    4. Opera
      Navegador con características avanzadas como VPN y bloqueador de anuncios integrado.
