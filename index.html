Ejercicio 1: HTML5



¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado HTML (HyperText Markup Language).
Es utilizado para estructurar y presentar contenido en la web. HTML5 introdujo varias mejoras y nuevas características
como soporte multimedia nativo, nuevos elementos semánticos y mejoras en el rendimiento.


Características principales de HTML5

  • Multimedia nativo
    • Soporte para video y audio sin necesidad de plugins.
  • Interactividad
    • API avanzadas para juegos y aplicaciones web interactivas.
  • Almacenamiento local
    • Permite el almacenamiento de datos en el navegador del usuario.
